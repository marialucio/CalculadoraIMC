# CALCULADORA IMC

Aplicação console para cálculo do IMC.

## Funcionalidades

- Calcula o IMC.
- Classifica o IMC.
- Calcula o peso que o usuário deve ganhar ou perder para chegar ao IMC ideal.

## Tecnologias utilizadas
 
- dotnet 6.0

## Como executar
 Realize o clone do repositório e execute o comando:
 
 ```
 dotnet run 
 ```
